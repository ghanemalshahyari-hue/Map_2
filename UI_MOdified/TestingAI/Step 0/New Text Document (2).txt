+# Step 1: Planning (التخطيط الأولي)
+
+## Overview
+
+Step 1 is the **Receipt of Mission** phase in the Military Decision-Making Process (MDMP). This step extracts preliminary data from uploaded source documents and generates initial planning documents for subordinate units.
+
+**Location**: `/opt/hermes/MODE_PRJOECT/src/pipeline/steps/step1_planning.py`
+
+---
+
+## Purpose
+
+The primary objectives of Step 1 are:
+
+1. **Extract preliminary information** from higher command's Warning Order and Intelligence Summary
+2. **Generate initial planning documents** to guide the planning process
+3. **Issue Warning Order #1** to subordinate units to initiate their preparation
+4. **Establish initial time estimates** for the planning cycle
+5. **Create initial staff brief** for command team awareness
+

+## Processing Behind Step 1
+
+### Execution Flow
+
+1. **File Upload**: User uploads Warning Order and Intelligence Summary via UI
+2. **Pipeline Initialization**: `run_langchain_pipeline_on_files()` is called
+3. **Parallel Chain Execution**: 16 LangChain chains run simultaneously
+4. **Data Extraction**: Each chain extracts specific data fields from source documents
+5. **JSON Generation**: Output saved to separate JSON files in `output/` directories
+6. **DOCX Generation**: Document generator creates formatted Arabic documents
+7. **Download**: Documents automatically downloaded to user's device
+
+### Key Function
+
+```python
+async_main(the_user_question_to_be_, brigade_input)
+```
+
+This function orchestrates the entire Step 1 pipeline:
+- Loads source documents
+- Executes 16 parallel LangChain chains
+- Processes structured output
+- Generates DOCX documents
+- Saves JSON files for potential downstream use
+
