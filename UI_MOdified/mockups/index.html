RMOOZ · Design alternatives

Alt 1 — Tactical

Vibe: ATAK / WinTAK. Pure black, monospace numerics, square corners.
Strong points: Classification banner top + bottom; reads as military immediately; high contrast for daylight tablets and gloves.
Trade-off: Less "premium" feeling than alt 2.

Alt 2 — Defense-Tech

Vibe: Anduril Lattice. Floating glass panels, command bar, soft shadows.
Strong points: Looks 2026-modern; command palette (Ctrl+K) front and center; map gets the most pixels because rail and panel float.
Trade-off: Glass + blur is pretty but heavier on weak GPUs / low-end tablets.

Alt 3 — Operations Console

Vibe: Palantir Gotham. Dense, dual side panels, amber accent, alerts and live activity.
Strong points: Built for analysts and TOC operators; selected feature + layers + activity are always visible; status bar top & bottom.
Trade-off: Map area is smallest of the 3; not ideal for tablets.
